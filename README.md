# AIアニメーションプレゼン自動生成システム

既存スライド画像からテーマカラーを自動抽出し、箇条書き入力だけでアニメーション付きプレゼンを自動生成するWebアプリです。

## ファイル構成

```
/
├── index.html       ← フロントエンド（テンプレート登録・入力・スライド表示）
├── api/
│   └── generate.js  ← Claude APIでスライド構成を生成するエンドポイント
├── package.json
└── README.md
```

## 機能

- **テーマカラー自動抽出**：既存スライド画像をアップロードするだけでメインカラーを自動解析
- **手動カラー指定**：カラーピッカーで色を手動指定も可能
- **AI構成生成**：Claude APIがスライド構成・本文テキストを自動生成
- **アニメーション表示**：タイトル・項目が順次アニメーションで表示
- **HTMLダウンロード**：生成したスライドをHTMLファイルとして保存・共有可能

## デプロイ手順

### 1. GitHubにリポジトリを作成

1. GitHub（https://github.com）を開く
2. 「New repository」でリポジトリを作成
3. 上記ファイルをすべてアップロード（api/フォルダごと）

### 2. Vercelに接続

1. Vercel（https://vercel.com）を開く
2. 「New Project」→ GitHubリポジトリを選択
3. 「Import」をクリック

### 3. 環境変数を設定

| 変数名 | 値 |
|--------|-----|
| `ANTHROPIC_API_KEY` | AnthropicのAPIキー |

※ APIキーは https://console.anthropic.com で取得

### 4. Deploy

「Deploy」をクリック → URLが発行されます

## 使い方

1. **STEP 1**：既存スライド画像をアップロード → テーマカラーが自動抽出される
2. **STEP 2**：テーマ・ターゲット・目的・箇条書きを入力
3. **STEP 3**：生成された構成を確認・必要なら修正
4. **STEP 4**：アニメーションスライドが完成 → HTMLでダウンロード

## 注意事項

- APIキーは絶対にGitHubにアップしないこと
- Vercelの環境変数にのみ設定すること
